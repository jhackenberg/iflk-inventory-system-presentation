1. Was das Inventarsystem ermöglichen muss

  • Schnelles Hinzufügen/Finden von Gegenständen
  • Eine übersichtliche, effiziente Verwaltung von größeren Datenmengen
  • Globale Erreichbarkeit der gleichen Daten durch alle Nutzer

2. Wie wir das umgesetzt haben

Ein Gegenstand:

  • Name
  • Seriennummer
  • Gebäude + Raum
  • Kategorie
  • Gruppen
  • Tags

Eine Kategorie:

... ist die grobe Art des Gegenstands
(z.B. Laptop)

  • Name
  • Icon?

Eine Gruppe:

... ist die genaue Art des Gegenstands
(z.B. MacBook Pro 2022)

  • Name
  • Icon?

Ein Tag:

... ist eine Zusatzinformation, die dem Gegenstand zugeordnet wird
(z.B. Größe: 16")

  • Wert
  • Icon?
  • Schlüssel?
  • Farbschema?

2. Wie wir das umgesetzt haben

Einfache und effiziente Bedienelemente: ☝

... ermöglichen schnelles Hinzufügen/Finden von Gegenständen

  • BarcodeScanner 📷
  • SearchBars 🔍
  • SammelInputs 📁

Performance 🚀:

... ermöglicht übersichtliche, effiziente Verwaltung von größeren Datenmengen

  • Svelte
  • GraphQL
  • Go
  • PostgreSQL

Die Cloud ⛅😎:

... ermöglicht generell fast alles (so auch das online Abrufen und Teilen von Daten)

  • Online von überall aufrufbar 🌐
  • Gleicher Datensatz für alle 🤝
